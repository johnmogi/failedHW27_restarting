    2. צרו פונקציה סינכרונית המקבלת לסוגריים מחיר ומחזירה ע"י return את המחיר כולל מע"מ (תוספת של 17%).

קיראו לפונקציה מלחצן שמקבל מחיר מהמשתמש ע"י prompt.

במידה ונזרקה חריגה – הציגו אותה ע"י alert.
